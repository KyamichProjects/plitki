Принцип работы

Как это работает

01
Выберите тип плитки
Прямая для прямых участков, клиновая — для поворотов. Каждая плитка имеет ширину 1000 мм.
02
Стройте дорожку
В конструкторе кликайте — каждая новая плитка стыкуется автоматически с зазором 146 мм.
03
Получите расчёт
Система автоматически подсчитывает количество плиток каждого вида и общую стоимость.
